# Identity verification support playbook

1. Never ask for SSN, account numbers, identity images, or notice access codes.
2. Confirm the issuing authority and notice type.
3. Link only to official verification channels.
4. Explain that processing can remain paused until verification is completed.
5. Escalate suspected identity theft.
