# Refund under review support playbook

1. Confirm federal versus state authority.
2. Confirm whether a notice was received.
3. Retrieve authority-specific review guidance.
4. If no action is requested, explain that review is ongoing and show the last update.
5. If a notice requests information, direct the user to the secure official response channel.
6. Escalate when the published timeframe is exceeded or financial hardship is reported.
