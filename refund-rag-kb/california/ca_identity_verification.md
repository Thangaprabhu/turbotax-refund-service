# California FTB Identity Verification

California FTB may issue notices to verify taxpayer identity and validate refund claims, especially when discrepancies or missing information are identified. Timely response is important to avoid refund delay or denial.

## User guidance

- If the user received a California FTB notice, advise them to follow the instructions on the notice.
- If identity verification is required, the user should respond through the official process indicated by FTB.
- If the user did not receive a notice, they should continue monitoring status and mail/email notifications.

## Assistant guardrail

Do not request identity documents in chat. Do not tell the user to call a non-official number.
