# California Refund Processing Timeframes

California FTB notes that some returns require extra review for accuracy, completeness, and fraud or identity-theft protection. These returns can take longer to process.

Typical published California timeframes include separate estimates for personal returns, business returns, amended returns, and refunds. Business refund processing can take longer than personal refund processing.

## User guidance

- Explain that California state processing can take longer if the return requires extra review.
- If the user filed a business return, set expectations that business refund processing may take longer than personal refund processing.
- If the user's refund is still within the expected processing window, advise checking again later.
- If the refund is outside expected timeframes, advise reviewing notices or contacting FTB through official channels.
