# California FTB Refund Status Guidance

California Franchise Tax Board provides a refund status service for taxpayers who may be owed a state refund. California refund processing can vary by filing method and whether the return requires extra review.

## User guidance

- For California state refunds, direct users to the California refund status flow or display the normalized status from the state refund integration.
- If a refund is processing, explain that state refunds are processed independently from federal refunds.
- If a federal refund is sent but California is still processing, reassure the user that this can happen because IRS and CA FTB systems are separate.

## Assistant guardrail

Do not mix California state guidance with IRS federal guidance unless clearly explaining the difference.
