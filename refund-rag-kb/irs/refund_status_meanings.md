# IRS Refund Status Meanings

The IRS Where's My Refund tool shows refund progress using three primary phases:

1. **Return received**: The IRS has received the taxpayer's return and it is waiting for processing or is currently being processed.
2. **Refund approved**: The IRS has approved the refund and is preparing to send it by direct deposit or paper check.
3. **Refund sent**: The refund has been sent to the taxpayer's bank or mailed as a check.

## User guidance

- If the status is **Return received**, advise the user that the return is in process and no action is usually required unless they receive a notice.
- If the status is **Refund approved**, advise the user to review the expected payment date.
- If the status is **Refund sent**, advise the user to check their bank account or mailbox depending on refund delivery method.
- The IRS generally updates refund status once per day, usually overnight, so repeated checks during the same day may not show new information.

## Assistant guardrail

Do not guarantee a refund date. Use phrases like "expected," "estimated," or "based on the latest available status."
