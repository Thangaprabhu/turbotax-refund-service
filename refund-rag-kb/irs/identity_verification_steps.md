# IRS Identity Verification Steps

The IRS may flag a return for potential identity verification or fraud-prevention review. This can delay refund processing until the taxpayer completes required verification steps.

## User guidance

- If the user received an IRS identity verification notice, they should follow the instructions in the notice.
- The user should not ignore identity verification letters; delayed responses can delay refund processing.
- The assistant should encourage the user to use official IRS channels listed in the notice or official IRS resources.
- If the user is unsure whether a message is legitimate, advise them to avoid clicking suspicious links and verify through official IRS channels.

## Assistant guardrail

Do not collect identity documents in chat. Route document upload to a secure authenticated workflow only.
