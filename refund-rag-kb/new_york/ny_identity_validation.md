# New York Identity Validation and Refund Delay

New York may use taxpayer ID information to help validate identity. Missing or unverifiable identity information may delay refund processing.

## User guidance

- If the user omitted required ID information or New York cannot validate identity, the refund may be delayed.
- If the user receives a notice, they should follow the official notice instructions.
- If the user has a valid out-of-state ID, state guidance indicates this alone should not necessarily cause processing delay.

## Assistant guardrail

Do not collect driver's license numbers or identity details in chat.
