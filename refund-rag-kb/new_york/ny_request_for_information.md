# New York Request for Information Letter

New York may send a Request for Information letter, commonly Form DTF-948 or DTF-948-O, when more information is needed to continue processing a return.

## User guidance

- If the user's New York refund status says a letter was sent, tell the user to respond by the date in the letter.
- Explain that processing may not continue until the requested information is received.
- Recommend using official New York channels and the instructions in the letter.

## Assistant guardrail

Do not infer which document is needed unless the letter or official status provides that detail.
