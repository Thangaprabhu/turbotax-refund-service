# Common Refund Delay Reasons

Refunds may be delayed due to:

- Return errors or incomplete information
- Missing signature
- Math error
- Identity verification
- Fraud prevention review
- Manual review
- Earned Income Credit or Additional Child Tax Credit review
- Amended return processing
- Injured spouse relief request
- Debt or refund offset
- Bank processing delay
- Paper return processing delay
- State-specific request for information
- Business return processing timelines

## User guidance

- Show the user their exact authority: IRS, CA FTB, NY DTF, or another state agency.
- Show the latest status and last updated time.
- If the user received a notice, they should respond by the notice deadline.
- If no notice exists and the refund is still within normal processing time, advise monitoring status.
- If outside normal processing time, recommend official support or authenticated support escalation.
